### Как запустить проект

Нужная версия java - 21;

В корневой директории проекта `../репозиторий/src/rogue-game`
выполнить команды

```
./gradlew build
```

```
./gradlew run
```


### Структура проекта
```
src/
└── main/
    └── java/
        └── org/rogue/
            ├── app/              # Точка входа, инициализация
            ├── datalayer/        # Работа с JSON файлами
            ├── domain/           # Чистая бизнес-логика (никаких внешних зависимостей!)
            └── presentation/     # JCurses, рендеринг, управление
              ├── view/           # Отрисовка
              ├── presenter/      # Связующее звено
```
